# LILA BLACK - Player Journey Visualization Tool

## Tech Stack

- Next.js 14 (TypeScript)
- Zustand (state management)
- Tailwind CSS (styling)
- Hyparquet (client-side parquet parsing)
- HTML5 Canvas (rendering)
- IndexedDB (local data persistence)

## Setup

```bash
# Clone the repository
git clone https://github.com/Paritoshsingh496/LILA-GAMES.git
cd LILA-GAMES

# Install dependencies
npm install

# Run development server
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build

```bash
npm run build
```

This generates a static export in the `out/` directory that can be deployed anywhere.

## Environment Variables

None required. The app is fully client-side with no backend or API keys.
